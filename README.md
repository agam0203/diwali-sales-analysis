# Diwali Sales Analysis

## Overview

This project analyzes customer purchasing behaviour during the Diwali festive season in India. The goal was to understand which product categories, states, age groups, and customer segments contributed the most to sales — and derive actionable insights from the data.

The dataset contains 1000 customer records with attributes such as gender, age group, state, occupation, product category, number of orders, purchase amount, and customer rating.

---

## Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Description

| Column | Description |
|---|---|
| Customer_ID | Unique identifier for each customer |
| Gender | Male or Female |
| Age_Group | Age bracket of the customer |
| State | State from which the purchase was made |
| Marital_Status | Married or Unmarried |
| Occupation | Customer's profession |
| Product_Category | Category of product purchased |
| Orders | Number of items ordered |
| Amount | Price per item (INR) |
| Rating | Customer satisfaction rating (3, 4, or 5) |
| Total_Revenue | Amount × Orders |

---

## Analysis Performed

- Revenue breakdown by product category
- State-wise sales performance (Top 7 states)
- Gender-wise spending pattern across categories
- Age group contribution to total revenue
- Occupation-wise purchase behaviour
- Average order value by marital status and gender
- Customer rating distribution

---

## Key Findings

- Electronics was the highest revenue-generating category during the festive season
- Delhi recorded the highest total sales among all states
- Customers in the 26–35 age group contributed the most to overall revenue
- IT Professionals had the highest spending compared to other occupations
- Married customers showed a higher average order value than unmarried customers
- Over 85% of customers gave a rating of 4 or 5, indicating strong satisfaction
